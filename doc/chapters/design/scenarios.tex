\section{Scenarios}\label{sec:scenarios}

In order to ease the analysis, we have defined three scenarios with different
values for \(A\), \(\frac{X}{Y}\) and \(N\). Inside each scenario, we will vary
all other factors and we will evaluate the impact of these factors in the
performances of the network. As said in \secref{sec:factors}, we will also vary
the number of users \(N\) but we will not consider this a tunable factor to
improve the performances of the network.

The scenarios defined are the following:
\begin{enumerate}
	\item \(A = 22500m^2\), \(\frac{X}{Y} = 1\) \idest{\(150m\times150m\)
		square} and \(N = 1125\mathit{users}\) \idest{\(\frac{N}{A} =
		\frac{1125\mathit{users}}{22500m^2} = 0.05\mathit{users}/m^2 =
		\frac{1}{20}\mathit{users}/m^2\)}. This is the ``high density
		scenario''.
	\item \(A = 250000m^2\), \(\frac{X}{Y} = 1\) \idest{\(500m\times500m\)
		square} and \(N = 1250\mathit{users}\) \idest{\(\frac{N}{A} =
		\frac{1250\mathit{users}}{250000m^2} = 0.005\mathit{users}/m^2 =
		\frac{1}{200}\mathit{users}/m^2\)}. This is the ``low density
		scenario''.
	\item \(A = 270000m^2\), \(\frac{X}{Y} = 3\) \idest{\(900m\times300m\)
		rectangle} and \(N = 1350\mathit{users}\) \idest{\(\frac{N}{A} =
		\frac{1350\mathit{users}}{270000m^2} = 0.005\mathit{users}/m^2
		\simeq \frac{1}{200}\mathit{users}/m^2\)}. This is the
		``rectangular scenario'' (low density).
\end{enumerate}
